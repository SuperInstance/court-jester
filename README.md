# court-jester

A creative AI tool for generating wild ideas, riffing on concepts, sharpening arguments, and provoking new perspectives — powered by DeepInfra.

## What This Gives You

- **Ideate** — generate 5 wild, practical, or absurd ideas on any topic
- **Riff** — creative riffing and brainstorming with style control
- **Sharpen** — hone arguments and find weak spots in reasoning
- **Provoke** — challenge assumptions with contrarian takes
- **Springboard** — launch from one idea into adjacent creative territory
- **Dialogue sessions** — multi-turn creative conversations with memory

## Quick Start

```bash
npm install court-jester

# Configure DeepInfra API key
export DEEPINFRA_API_KEY=your_key_here

# Ideate on a topic
court-jester ideate --topic "sustainable fishing" --style wild --count 5
```

### Tools

| Tool | Description |
|---|---|
| `ideate` | Generate N ideas (wild/practical/absurd) with pros/cons |
| `riff` | Freeform creative brainstorming |
| `sharpen` | Strengthen arguments, find weaknesses |
| `provoke` | Challenge assumptions contrarian-style |
| `springboard` | Jump to adjacent creative territory |
